# 📚 bookSearchApp
## 앱 개발 심화 주차 과제 - Advance (책 검색 앱 만들기)
**책 검색하고 저장하는 앱** 

> CoreData와 HTTP 통신을 사용하여 책을 검색하고 저장하는 앱의 기능을 구현합니다.
